# Source Inputs

Bulky raw source inputs are not committed by default.

The investigation used source files such as ODHF v1.1, ODHF metadata, and Montreal boundary data. The processed evidence needed for the project scaffold is committed under `data/datasets/` and `data/reports/`.

When raw source inputs are needed, retrieve them from the official source listed in:

- `data/reports/Consolidated-Data-Investigation-Report.md`
- `data/datasets/source_manifest.csv`
- `data/datasets/navigator_source_manifest.csv`

Do not surface ODHF-derived facility records in the app without source name, publisher, source URL, license or terms reference, retrieval date, source record ID where available, and last-verified date.
